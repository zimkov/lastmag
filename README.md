# Методы искусственного интеллекта

## Метод Гаусса для решения СЛАУ

> Система линейных алгебраических уравнений (СЛАУ) — это набор уравнений, для которых нужно найти значения переменных, удовлетворяющие всем уравнениям одновременно. То есть искомые значения должны быть верны не для одного уравнения по отдельности, а для всей системы целиком.

> Метод Гаусса — это способ решения систем линейных алгебраических уравнений. Его идея в том, что исходную систему уравнений записывают в виде матрицы — числовой таблицы со строками и столбцами. Такой подход позволяет работать только с коэффициентами и свободными членами, не оперируя буквенными обозначениями.

## Расчет определителя

> Определитель (детерминант) — это числовая характеристика квадратной матрицы, которая содержит важную информацию о матрице.

Основные свойства:

- Определитель существует только для квадратных матриц (n×n)
- Если определитель = 0, матрица называется вырожденной (не имеет обратной)
- Если определитель ≠ 0, матрица невырожденная (имеет обратную)
- Определитель показывает, во сколько раз матрица "растягивает" пространство

Формулы для маленьких матриц:

Для матрицы 2×2:

$$ \begin{vmatrix} a & b \\ c & d \end{vmatrix} = ad - bc $$

Для матрицы 3×3 (правило Саррюса):

$$ \begin{vmatrix} a & b & c \\ d & e & f \\ g & h & i \end{vmatrix} = aei+bfg+cdh−ceg−bdi−afh $$

## Интерполяция Лангража

> Интерполяция Лагранжа — это математический способ построить единую плавную кривую (полином), которая проходит строго через все заданные вами точки. Зная уравнение этой кривой, можно вычислить значение в любой промежуточной точке.

Если у нас есть $n+1$ точек $(x0,y0),(x1,y1),...,(xn,yn)$, то полином Лагранжа $L(x)$ вычисляется так:

$L(x) = \sum_{j=0}^n y_j * l_j(x)$

Где $l_j(x)$ — это так называемые **базисные полиномы**. Их главная "магия" в том, что:

- $l_j(x)$ равен **1**, если $x$ совпадает с $x_j$​.
- $l_j(x)$ равен **0**, если xx совпадает с любой другой точкой $x_i$ (где $i≠j$).

Формула базисного полинома выглядит так:

$l_j​(x) = ∏_{i=0,i ≠ j}^n  ​( x−x_i ) / (​ x_j​−x_i )$
